# Clubes famosos subset

Generated with `node scripts/create-famous-clubs-db.mjs`.

This subset keeps players whose `team` matches selected major clubs available in the EA FC26 source CSVs. Real Madrid and FC Barcelona are both listed under `LALIGA EA SPORTS` in this dataset.

Included clubs: Arsenal, AS Roma, Atlético de Madrid, Borussia Dortmund, Chelsea, FC Barcelona, FC Bayern München, FC Porto, Juventus, Liverpool, Manchester City, Milano FC, Paris SG, Real Madrid, SL Benfica, Sporting CP, Spurs, and SSC Napoli.
